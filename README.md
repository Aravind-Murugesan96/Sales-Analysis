# 📊 Sales Analysis Project

## 📌 Project Overview

This project analyzes sample sales data for AAL, a clothing company in Australia, to identify high-performing states and customer groups.

## 🎯 Objectives

* Identify highest and lowest revenue states
* Analyze sales across customer groups (Kids, Women, Men, Seniors)
* Perform time-based analysis (daily, weekly, monthly, quarterly)
* Provide business recommendations

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Seaborn
* Matplotlib
* Jupyter Notebook

## 🔍 Project Steps

### 1. Data Wrangling

* Checked missing values using `isna()`
* Handled missing data using appropriate techniques
* Applied normalization

### 2. Data Analysis

* Performed statistical analysis:

  * Mean
  * Median
  * Mode
  * Standard Deviation
* Identified:

  * Highest & lowest sales by state
  * Highest & lowest sales by group

### 3. Data Visualization

* State-wise sales analysis
* Group-wise comparison
* Time-of-day sales trends
* Weekly, Monthly, Quarterly insights
* Used Seaborn for statistical visualization


### 5. Recommendations

* Focus marketing on low-performing states
* Run targeted campaigns during peak hours
* Improve inventory in high-demand regions
